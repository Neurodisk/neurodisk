🔐 Activer la Sécurité du compte

Clinique Neurodisk — Sécurité des dossiers (Loi 25) · Obligatoire pour les administrateurs et professionnels

Pourquoi ? La double authentification ajoute un code à usage unique en plus de votre mot de passe. Même si quelqu'un découvrait votre mot de passe, il ne pourrait pas accéder aux dossiers sans votre téléphone. C'est exigé pour protéger les renseignements de santé.

Étape 0 — Installez une application d'authentification sur votre téléphone (gratuit) :
Google Authenticator · Microsoft Authenticator · Authy. Téléchargez-la depuis l'App Store ou Google Play.

1. Connectez-vous au site

  1. Allez sur https://neurodisk.gabrielgirard-kin.workers.dev
  2. Entrez votre courriel et votre mot de passe habituels.

2. Ouvrez la section Sécurité

  1. Administrateurs : dans le panneau d'administration, cliquez sur le bouton 🔐 Sécurité en haut à droite.
  2. Professionnels : dans votre bibliothèque, cliquez sur le bouton 🔐 Sécurité en haut à droite.

3. Scannez et activez

  1. Un QR code s'affiche. Ouvrez votre application d'authentification et choisissez « Scanner un code ».
  2. Scannez le QR code à l'écran. Une entrée « Neurodisk » apparaît dans l'app avec un code à 6 chiffres qui change toutes les 30 secondes.
  3. Entrez ce code à 6 chiffres dans le champ, puis cliquez Activer la sécurité.
  4. C'est fait ! ✅
Important — gardez votre clé de secours. Sous le QR code se trouve une « clé manuelle ». Notez-la et conservez-la en lieu sûr : elle permet de reconfigurer la sécurité si vous changez ou perdez votre téléphone.

4. Aux prochaines connexions

Après votre mot de passe, le site vous demandera le code à 6 chiffres affiché dans votre application. Entrez-le pour accéder aux dossiers.

Téléphone perdu ou bloqué ? Contactez votre administrateur. Il peut réinitialiser votre sécurité pour que vous puissiez vous reconnecter et la reconfigurer.
Clinique Neurodisk — Procédure interne de sécurité · Conforme aux exigences de la Loi 25 sur la protection des renseignements personnels.
